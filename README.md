# es-head-plugins


> This is a elasticsearch-head plugin for mac chrome 

1. download

```
$ git clone https://github.com/llmgo/es-head-plugins.git
```

2. unarich

```
$ cd es-head-plugins
$ unzip elasticsearch-head.zip
```

3. open chrome input

```
$ chrome://extensions/
```
